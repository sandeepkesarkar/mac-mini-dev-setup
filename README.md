# Mac Dev Setup

A complete, opinionated guide to setting up a Mac Mini as a headless development machine — controlled entirely from a MacBook Pro.


## Who This Is For

- Developers setting up a Mac Mini (or Mac Studio/Mac Pro) as a dedicated dev server
- Anyone who wants to control one Mac from another Mac
- Developers looking for a modern, productive terminal setup

## Guides

| # | Guide | What You'll Set Up |
|---|-------|--------------------|
| 1 | [System Preparation & Remote Access](guides/01-system-preparation.md) | Energy settings, auto-login, SSH, Screen Sharing, Tailscale, Luna Display, Jump Desktop |
| 2 | [Terminal Setup](guides/02-terminal-setup.md) | Homebrew, iTerm2, Oh My Zsh, Powerlevel10k, plugins, modern CLI tools |
