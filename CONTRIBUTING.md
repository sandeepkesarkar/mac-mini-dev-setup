# Contributing

Thanks for your interest in improving these guides!

## How to Contribute

**Found a mistake or outdated instruction?**
Open an issue describing what's wrong and which guide it's in.

**Want to add something?**
Fork the repo, create a branch, and submit a pull request. Please keep the same tone and formatting style as the existing guides.

## Guide Style

- Use step-by-step numbered instructions for sequential tasks
- Include the exact commands to run, with copy-pasteable code blocks
- Note any gotchas or platform-specific caveats (Intel vs Apple Silicon, macOS version differences)
- Add a verification step where possible so readers can confirm things worked
- Keep it practical — explain the "why" briefly, but focus on the "how"

## Adding a New Guide

1. Create a new markdown file in `guides/` with the naming pattern `XX-topic-name.md`
2. Add it to the table in `README.md`
3. Link to adjacent guides with relative paths where relevant
